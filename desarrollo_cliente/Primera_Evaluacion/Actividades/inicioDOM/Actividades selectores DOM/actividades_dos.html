Actividades modificación apariencia y comportamiento

Objetivos: Aprender a seleccionar elementos del árbol DOM y modificar su apariencia y comportamiento.

Duración estimada: 3 horas.

Recursos:

• W3schools

• MDN

• DevDocs

  • Continuar

Ejercicio 1:

En la consola del navegador y utilizando document.documentElement, document.body, document.head, document.title y parentNode, previousSibling, nextSibling, firstChild, lastChild y childNodes, navega hasta llegar a:
- El cuerpo del archivo HTML y muestra sus nodos hijos.
- El tercer hijo del cuerpo del HTML.
- El sidebar.
- El ejercicio 3 del sidebar.
- El título de segundo nivel de este ejercicio.

Ejercicio 2:

En la consola del navegador y empleando ahora también los selectores getElementById, getElementsByTagName, getElementsByName, getElementsByClassName, querySelector, querySelectorAll, navega hasta llegar a:
- El sidebar.
- El ejercicio 3 del sidebar.
- El título de segundo nivel de este ejercicio.

- Este párrafo subrayado.

- Este párrafo en cursiva y subrayado.
