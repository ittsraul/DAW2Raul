Actividades creación dinámica de código HTML desde JavaScript

Objetivos: Aprender a crear HTML de forma dinámica desde JavaScript.

Duración estimada: 3 horas.

Recursos:

• W3schools

• MDN

• DevDocs

Ejercicio 1:

Implementa un script para que en el centro del archivo HTML proporcionado se visualicen tres imágenes con tres textos sobreimpuestos: 'café', 'infusions' y 'alcohol'.

Ejercicio 2:

Implementa un script que permita que al pulsar sobre las imágenes implementadas en el script anterior, éstas se eliminen y sean reemplazadas por las cinco imágenes correspondientes a cada apartado, incluidas en la carpeta src proporcionada -hay 5 imágenes para cada categoría: café, infusions y alcohol-.

Ejercicio 3:

Completa el script anterior para que al pulsar sobre cualquiera de las cinco imágenes de cada categoría, éstas se eliminen y se muestre únicamente la imagen pulsada, ocupando el mismo espacio que ocupaban las cinco imagenes.
